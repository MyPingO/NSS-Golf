About NSS Golf Shots Gallery

Welcome to the NSS Golf Perfect Shots Gallery project! This is an open-source web application that allows users to view and share their perfect score shots for Nintendo Switch Sports Golf. Perfect score shots are Hole in One (Par 3), Eagle (Par 4), and Albatross (Par 5). Users can view the images in a grid card format on the gallery page, or can search for specific conditions on the search page. Every image can be clicked to view in a larger size and contains associated video links for further content. You can submit an image to the site to exapnd the gallery and it will appear on the site after an Admin has approved it. Admins have the ability to delete images if necessary.

Usage Instructions

  1. Register/Log In: If you're a new user, register an account. If you're a returning user, you should be logged in automatically. If not, log in here!
  2. Browse the Gallery: The gallery page is where you can see all the images uploaded to the site. Click on an image to view it in a larger size. Each video has its associated video link (YouTube, Vimeo, etc.).
  3. Browse Community TutorialsIf you want to become better at NSS Golf, consider checking out the tutorials page.These videos are made by the community, for the community!
  4. Uploading a Shot: If you are logged in, you can upload your shots! If you make a mistake, don't worry! An Admin will review the submission and edit it if necessary before accepting it. If even the Admin can't fix your submission, it will be rejected and the next time you log on to the site, you'll see a message explaining how to resubmit your shot!
  5. Search for a Shot: If you're looking for a specific shot, or just want to filter the submissions, you can do so here!

Additional Information

This project is built with Flask, a lightweight WSGI web application framework. It's designed to make getting started quick and easy, with the ability to scale up to complex applications. Learn more about Flask.
